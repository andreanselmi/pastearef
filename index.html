PasteARef

PasteARef is a embedded SketchFab model viewer built on the Sketchfab Viewer API with more custom controls for digital and traditional artists looking for references!

Looking for more info? Check out the GitHub page. Check the Sketchfab Docs.

Model lighting
Sketchfab allow models to have up to 3 "analytical" real-time lights. These can be fully customized from the Viewer API. See here for more info. They can also be rotated around the model! Hold Alt + drag the mouse.
